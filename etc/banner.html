CSV Data Validator
The Kohesio data specification is currently in development. As such the current service is offered as a beta version.

This service allows you validate your data against the CSV data import format expected by Kohesio. Access to this service is anonymous and no data, nor validation reports, are retained following its use. The service is also offered via SOAP API to facilitate machine-to-machine integration.
